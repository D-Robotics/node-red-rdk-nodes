# node-red-node-rdk-tools
在Node-RED中配合地平线RDK硬件使用的工具节点。

## 安装
可以在Node-RED编辑器中的节点管理面板中进行安装。也可以进入node-red安装目录进行手动安装：
```
    cd ~/.node-red
    npm i node-red-node-rdk-tools
```
